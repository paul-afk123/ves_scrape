# ves_scrape

**Funnel Finder** – find ad landing pages and funnel paths from any website.

---

## Open the app (live)

**[→ Open Funnel Finder app](https://ves-scrape.vercel.app)**

*(If the link doesn’t work yet, deploy once with the button below; Vercel will give you a URL like `https://ves-scrape.vercel.app` or `https://ves-scrape-xxx.vercel.app`.)*

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/paul-afk123/ves_scrape)

**Deploy steps:** Click the button above → sign in with GitHub → set **Root Directory** to `funnel-finder` → Deploy. Your live app URL will appear in the Vercel dashboard.
